## 二、E4 ALS 收编

- `als-xy-wing`（difficulty 840）保留独立 strategyId，内部委托 `searchALSChain(..., { exactLinks: 2 })`。
- `als-chain`（880）为 canonical owner，`exactLinks` 未设、`minLinks: 2` 覆盖更长链。
- overlap family `als-chain` 已登记两者；行为：短链仍优先以 `als-xy-wing` 名命中（更低 difficulty），长链由 `als-chain` 兜底。

## 三、健全性修复（验收关键）

1. **bug-plus-n**：BUG 数须在行/列/宫均为奇；近 BUG 网格要求其余空格双值且各宫偶计数；多格非双值仅在三候选单格时落子。
2. **multi-coloring Type1**：弱链须跨色（A–B 簇），且非共轭对；双簇不用 sector link。
3. **3d-medusa**：R4/R5 改为「同一宫内」见双色；矛盾规则（R1/R2/R6）暂不发射落子/整色消去，只保留离链消去。
4. **aic-with-als/ur**：`resultToStep` 剥离 placements，仅输出 eliminations。
5. **tridagon**：暂仅 Type1（恰好 1 个守护格，其余 11 格候选 ⊆ D）。

## 五、难点与处理

| 难点 | 处理 |
|---|---|
| Multi-Colors 弱链误判 | 要求 `col1 ≠ col2` 且非 conjugate；禁用 sector 分簇 |
| 3D Medusa 整色落子级联错误 | 先只实现 R3–R5；矛盾落子待分量内「每格至多一真色候选」校验后再开 |
| BUG+N 奇偶判定过宽 | 改为三宫同时奇 + `isNearBugGrid` |
| 扩展 AIC 部分盘面多解 | placements 禁用，仅 eliminations + `isSound` 过滤 |
| Tridagon Type2 误报 | 本期仅 Type1；Type2 待共同见证格验证后恢复 |
