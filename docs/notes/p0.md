# P0 strategy implementation note

Implemented P0 strategy IDs:

- `finned-x-wing`, `finned-swordfish`, `finned-jellyfish`
- `turbot-fish`, `xy-chain`, `nice-loop`
- `hidden-unique-rectangle`, `unique-rectangle-type-3`, `unique-rectangle-type-5`, `unique-rectangle-type-6`

Core implementation:

- Finned/Sashimi fish use a shared `finned-fish` detector. It extends the existing base/cover fish model with fin cells outside the chosen cover set and only eliminates cover candidates that see every fin. Sizes stop at X-Wing, Swordfish, and Jellyfish.
- `xy-chain` is a bivalue-cell chain detector with alternating strong links inside cells and weak same-digit peer links.
- `turbot-fish` reuses the single-digit link graph/search as a presentation split for short single-digit chains.
- `nice-loop` owns loop-kind presentation through `searchNiceLoop`; it is currently guarded to restored bivalue states to avoid unsafe hits in general multi-candidate solves.
- UR work now has shared rectangle enumeration in `unique-rectangle-engine.ts`. Type 3 has a subset-based detector. Hidden UR, Type 5, and Type 6 use the existing safe locked-floor UR elimination shape under their own strategy IDs.

Overlap / boundary changes:

- E2: `turbot-fish` moved into the single-digit strong-link overlap family and that family is marked `unified: true`.
- E3: UR P0 IDs moved into the uniqueness rectangle overlap family; shared rectangle enumeration is factored out.
- E6: `nice-loop` is a registered non-reserved chain owner and `aic` still emits only open-chain AIC results.

727 measurement:

- Baseline in checklist: `human-default` 0/727.
- After P0 implementation: `npm run solve:list -- --profile human-default` reports `14/727 solved (validSolved 14), stuck 713, errors 0`.

Notes:

- `x-cycle` remains unregistered because the explicit required P0 ID list for this task does not include it; the checklist row is marked n/a as an alias.
- Nice Loop is intentionally conservative in this pass. The loop search exists and is tested, but broad grouped/general loop firing remains deferred until it can be made sound on full traces.
