# M2 Design Notes

## Implemented Strategies

- T1 singles: added Full House and Hidden Single alongside the existing Naked Single. Both produce placements only.
- T2 intersections: added Pointing and Claiming as separate strategies. They find box-line or line-box locked candidates and return candidate eliminations.
- T2 subsets: added Naked Pair/Triple/Quad and Hidden Pair/Triple/Quad. Naked subsets scan N cells whose union is N digits; hidden subsets scan N digits confined to N cells.
- T3 fish: added X-Wing, Swordfish, and Jellyfish through a shared base/cover implementation parameterized by fish size.
- T3 single-digit patterns: added Skyscraper, 2-String Kite, and Empty Rectangle as conservative short strong-link patterns.
- T3 wings: added XY-Wing, XYZ-Wing, and W-Wing using bivalue/trivalue cell templates and explicit peer checks.

## Key Design Decisions

- Each named strategy has its own module under `packages/engine/src/strategies/` and exports a `Strategy` object. Shared mechanics live in `strategies/utils.ts`; this keeps strategy modules small without changing engine foundations.
- Fish use the required unified base/cover model. Rows and columns are both tried as base sets; X-Wing/Swordfish/Jellyfish differ only by size and display metadata.
- Subsets use small combination scans over houses. The search space is bounded by 9 cells or 9 digits, so a simple implementation is clear and fast enough.
- Turbot-family strategies use a generic two-strong-link plus weak-link helper, with per-strategy geometry predicates. This avoids duplicating chain mechanics while keeping names human-readable in trace output.
- T3 recognizers are intentionally conservative. They return a step only when all pattern cells and eliminations are explicit; ambiguous or advanced variants are left for M3 chains/AIC.

## Difficulties and Handling

- Hand-built tests for chain-like patterns can accidentally invalidate the intended strong link by adding an elimination candidate in the same house. The tests were adjusted to keep conjugate pairs exact.
- Hidden subset strategies can overlap hidden singles. To preserve the simpler-first flow, hidden subsets require each selected digit to appear at least twice in the house.
- Empty Rectangle has several human variants. The M2 implementation uses a conservative box-line strong-link form rather than broader grouped inference; broader forms belong naturally in M3 grouped chains.
- Soundness was prioritized over maximum solve rate. The full 400-puzzle ground-truth regression passes with zero trace violations.
