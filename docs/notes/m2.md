# M2 设计说明

## 实现的策略
- T1: hidden-single (difficulty 10)
- T2: locked-candidates (pointing/claiming, difficulty 20), naked/hidden pairs/triples/quads (30)
- T3: fish (X-Wing/Swordfish/Jellyfish 使用统一 base/cover 模型, 40), single-digit patterns (Skyscraper/2-String Kite/Empty Rectangle, 45), wings (XY/XYZ/W-Wing, 50)

## 核心思路与设计决策
- 所有策略独立模块, export Strategy 对象, apply 不 mutate grid, 首 个适用 step 即返回。
- 鱼类使用统一 base/cover 集合模型: base 是候选所在的行/列集合, cover 是消除所在的列/行集合; size=2/3/4 对应 X/Sword/Jelly。
- 单数字模式和 wings 分别独立实现, 利用 bivalue/trivalue 候选计数。
- 数据结构: 主要用 bitmask + HOUSES 遍历, 避免复杂对象以保持性能和简洁。
- 术语严格对齐 glossary.zh-en.md (e.g. 隐性唯一, 区块排除, X翼)。
- 策略顺序按 difficulty 升序, 保证便宜优先 (FR-7)。
