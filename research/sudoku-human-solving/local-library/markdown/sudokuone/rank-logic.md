# SudokuOne Rank Logic / Universal Logic

Source ID: `SUDOKUONE-RANK-LOGIC`

URL: https://sudoku.allanbarker.com/sweb/general.htm

Accessed: 2026-06-23

Local research summary:

- Allan Barker's SudokuOne material describes a universal logic for Sudoku using base sets/truths and cover sets/links.
- Rank is defined as `number of cover links - number of bases`.
- Rank 0: any non-base candidate in a link can be eliminated.
- Rank 1: any non-base candidate contained in two links can be eliminated.
- Rank R: any non-base candidate contained in `R + 1` links can be eliminated.
- The page relates rank logic to fish, chains, finned fish, Kraken fish, odd loops, broken wings, and dark/illegal logic.
- Odd loops and broken wings are treated as illegal/dark logic; guardians become a virtual set/truth.
