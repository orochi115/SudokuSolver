# XY/XYZ/W Wings / 翼类技巧

## One-Sentence Rule

Wing patterns are short, memorable chain templates built from bivalue or trivalue cells.

## XY-Wing

Pivot `{X,Y}` sees pincers `{X,Z}` and `{Y,Z}`. Candidate `Z` is removed from cells seeing both pincers.

## XYZ-Wing

Pivot `{X,Y,Z}` sees pincers `{X,Z}` and `{Y,Z}`. Candidate `Z` is removed from cells seeing the pivot and both pincers.

## W-Wing

Two bivalue cells with the same pair are bridged by a strong link on one digit. The other digit is removed from cells seeing both bivalue cells.

## Formula Role

Wing patterns can be treated as human-friendly abbreviations for short chains. If the direct wing has no elimination, it may still create a useful strong inference for a constructed chain.
