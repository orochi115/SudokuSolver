# Fish Base/Cover Model / 鱼类基础-覆盖模型

## One-Sentence Rule

For one digit, if N base houses' candidates are fully covered by N cover houses, eliminate that digit from cover-house cells outside the base houses.

## Names by Size

- Size 2: X-Wing / X翼.
- Size 3: Swordfish / 剑鱼.
- Size 4: Jellyfish / 水母.

## Human Scan Procedure

1. Pick one digit.
2. Find rows or columns where the digit has two to four candidate positions.
3. Group N base rows/columns whose candidates fall in exactly N cover columns/rows.
4. Remove the digit from cover houses outside the base intersections.

## Formula Role

This model is more general than memorizing fish names. It can unify X-Wing, Swordfish, Jellyfish, and some advanced fish variants.
