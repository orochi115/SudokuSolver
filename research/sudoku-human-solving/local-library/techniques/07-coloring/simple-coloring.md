# Coloring / 染色法

## One-Sentence Rule

Coloring visualizes conjugate-pair chains for one digit; contradictions or cells seeing both colors create eliminations.

## Human Scan Procedure

1. Pick one digit.
2. Find conjugate pairs for that digit.
3. Assign opposite colors across each strong link.
4. Use Color Trap: an uncolored candidate seeing both colors can be removed.
5. Use Color Wrap: if two same-colored candidates see each other, that color is false.

## Formula Role

Coloring is not separate from chains; it is a visual method for finding single-digit AIC/X-chain eliminations.
