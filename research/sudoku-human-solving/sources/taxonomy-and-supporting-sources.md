## Sudopedia Taxonomy
Sudopedia gives a concise community taxonomy of solving techniques:
- Singles: full house, last digit, hidden single, naked single.
- Intersections: locked candidates, pointing, claiming, locked pair/triple, almost locked candidates.
- Subsets: naked and hidden pairs/triples/quads.
- Fish: X-Wing, Swordfish, Jellyfish, Squirmbag, finned, sashimi, franken, mutant, kraken.
- Single digit patterns: Skyscraper, 2-String Kite, Empty Rectangle.
- Coloring: simple colors, multi-colors, weak colors, X-colors, color trap/wrap/wing, 3D Medusa.
- Uniqueness: uniqueness tests, avoidable rectangle, BUG, BUG Lite, uniqueness controversy.
- Chains and loops: forcing chain, X-Chain, XY-Chain, remote pairs, X-Cycle, broken wing, nice loops, DIC, AIC, oriented chains.
- Wings: XY-Wing, XYZ-Wing, WXYZ-Wing, W-Wing.
- ALS: ALS-XZ, ALS-XY-Wing, ALS-XY-Chain, Death Blossom.
- Enumeration on selected cells: aligned pair/triple/subset exclusion.
- Miscellaneous: Sue de Coq, subset counting, braid analysis, constraint subsets, equivalence marks.
- Last resort: forcing net, tabling, GEM, Bowman Bingo, trial and error, Nishio, templating, guessing.
- Brute force: backtracking algorithms, primarily for computer solvers.

## Sudoku of the Day
- Presents a simpler teaching-oriented sequence: Single Position, Single Candidate, Pencilmarks, Candidate Lines, Double Pairs, Multiple Lines, Naked Pairs/Triples, Hidden Pairs/Triples, X-Wing, Swordfish, Forcing Chains, Nishio, Guessing.
- It explicitly says most newspaper Sudoku can be solved with easy techniques.
- It distinguishes forcing chains from guessing: forcing chains inspect both alternatives and act only on the common result.
- It also states that some puzzles may require guessing, which conflicts with stricter logic-only sources and should be treated as a practical teaching-site view rather than a formal completeness result.

## Sudoku.com Pointing Pairs
- A beginner-friendly explanation: if a note/candidate appears twice in a block and both appearances lie in the same row or column, the candidate must occupy one of those block cells, so it can be eliminated from the rest of the row/column.
- Useful for plain-language Chinese/English explanations, but not enough for advanced taxonomy.

## Sudoku Bliss
- Provides recent teaching-oriented advanced summaries.
- Advanced list includes W-Wing, XY-Wing, XYZ-Wing, X-Wing, Swordfish, Skyscraper, Unique Rectangle, Locked Candidates, ALS-XZ, Forcing Chains, X-Cycles.
- Useful for practical scanning steps and “best for / key components / required link / elimination / failure conditions” framing.
- ALS page reinforces the formula `N cells with N+1 candidates`, RCC vs unrestricted common candidate, ALS-XZ Type 1/2, and ALS-XY-Wing.

## Contribution to Formula-Like Workflow
- Sudopedia is the broadest index and helps ensure coverage completeness.
- Sudoku of the Day helps create a beginner-to-master teaching order but is less rigorous at the extreme end.
- Sudoku Bliss provides useful “how to find” checklists that can be converted into procedural scans.
- Sudoku.com is useful for simple explanations of common techniques.
